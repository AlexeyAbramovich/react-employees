# Functionality
- User Registration
- User Athorization by JWT token
- Private routes
- Every user can add employees
- Every user can edit/remove self-added employees
- Every user can look at list of all employees
- User can't edit/remove other's users employees

# Used technologies
## Frontend
- TypeScript
- Redux Toolkit
- RTK Query
- antd
- react-router-dom
## Backend
- JS
- Express
- Prisma
- concurrently
- dotenv
- bcrypt
- jsonwebtoken
