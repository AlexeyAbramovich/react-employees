# Functionality
- User registration
- User authorization by JWT token
- Form validation
- Private routes
- Every user can add/edit/remove its own employees
- Every user can look at list of all employees
- User can't edit/remove other's users employees

# Used technologies
## Frontend
- React
- TypeScript
- Redux Toolkit
- RTK Query
- Create-React-App
- antd
- react-router-dom
- HTML
- CSS
## Backend
- JS
- Express
- Prisma
- concurrently
- dotenv
- bcrypt
- jsonwebtoken
